Hi, I’m Jasmine.

My current home is Pittsburgh, Pennsylvania.

Background

I am an aspiring Front End Web Developer!

My journey started when I was 13-years-old at my local library where I would spend hours using HTML & CSS (aka copy & paste) to create blogs and build guilds on Neopets.

I drifted away from the coding days of my youth, but found myself returning to creation as I began projects as a health coach and wellness advocate.

I realized that all of my wellness endeavors involved one of my earliest passions: creating websites and online communities. So here I am learning to code!

Goals

I want to master the process of building web sites with a thoughtful and responsive design by increasing my knowledge, skills and abilities in:

  • Web Design
  • HTML
  • CSS
  • JavaScript
  • React

I’d like to work in the health and wellness industry to help professionals develop responsive and accesible websites and applications.
